# Правда или ложь? Определение достоверности / недостоверности информации по мультимодальному тексту

## Краткое описание проекта
Этот репозиторий содержит материалы, разметку и код, разработанные в рамках проекта НУГ «КомпЛинг» — научно-учебной группы Центра языка и мозга НИУ ВШЭ, направленного на решение задачи определения уровня достоверности информации по устной речи, мимике и кинесике. nnov.hse.ru (https://nnov.hse.ru/human/neurolingcentre/compling/project?utm_source=chatgpt.com)

Цель проекта — создание корпуса мультимодальных данных и разработка методов автоматизированного определения правдивости/ложности высказываний на основе анализа речи, голосовых характеристик, мимики и жестов.
